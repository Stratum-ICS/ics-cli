## Key Decisions Made

| Decision | Rationale | Alternatives Considered |
|----------|-----------|------------------------|
| **Rust** (edition 2021) | Single binary, crates.io `ics-cli`, aligns with user direction | Python (earlier draft) |
| **Hybrid B0 store** | Blobs on disk + SQLite for DAG/refs | Flat files only; SQLite-only |
| **CLI `checkout`** for restore-from-HEAD | Matches git mental model; avoids confusion with `git revert` | Name `revert` in spec (superseded) |
| **OpenAPI pinning** | `tests/fixtures/` or checksum in `routes.rs` header; not default-commit from `.claude/tmp` | Ephemeral curl-only |
| **Refs in SQLite** | Flat names **`HEAD`** and **`main`** (documented) | `refs/heads/main` strings |
