# B1 — Stratum HTTP integration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extend **`ics`** so an authenticated user can sync markdown files with Stratum **notes** (create/update/publish as required by the wiki) and create/submit **proposals** via `POST /api/proposals`, maintaining Stratum IDs in **YAML frontmatter** or a fallback index file per design spec §3.2.

**Architecture:** Thin **`StratumClient`** (`httpx.Client`) with injectable base URL; auth tokens loaded/stored under `~/.config/ics-cli/` (`0600`). Domain helpers convert workspace paths ↔ Stratum note payloads. CLI adds subcommands (`login`, `push`, `proposal create`, `proposal submit`, … — exact names in Task 5).

**Tech Stack:** Python ≥3.10, **httpx**, **Typer**, **PyYAML** (frontmatter). Tests use **`unittest`** + **`httpx.MockTransport`** for deterministic responses.

**Prerequisite:** [`2026-05-05-ics-cli-b0.md`](2026-05-05-ics-cli-b0.md) complete on branch.

### Task 2: Stratum HTTP client (mocked)

**Files:**

- [ ] **Step 1: Define `StratumClient`** with methods matching pinned routes: at minimum `login(...)`, `create_or_update_note(...)`, `publish_note(...)` if separate in OpenAPI, `post_proposal(body: dict)`.

Example **test** shape (adjust URLs to match `routes.py`):

```python
import json
import unittest

import httpx

from ics_cli.stratum.routes import POST_PROPOSALS
from ics_cli.stratum.client import StratumClient


class TestStratumClient(unittest.TestCase):
    def test_post_proposal_success(self):
        def handler(request: httpx.Request) -> httpx.Response:
            self.assertEqual(request.url.path, POST_PROPOSALS)
            return httpx.Response(200, json={"id": "p1", "status": "pending"})

        transport = httpx.MockTransport(handler)
        client = StratumClient(base_url="http://test", token="t", transport=transport)
        out = client.post_proposal({"team_id": 1, "note_ids": [2], "rationale": "x" * 50})
        self.assertEqual(out["id"], "p1")
```
